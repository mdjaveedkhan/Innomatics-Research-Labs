# 🧠 GenAI Prompt Engine using LangChain

This project focuses on building a dynamic and reusable prompt system using LangChain instead of hardcoded prompts.

The goal is to design flexible prompt templates that can adapt to different inputs, styles, and use cases.

---

## 📌 Project Overview

In real-world Generative AI applications, prompts should not be hardcoded.  
This project demonstrates how to create reusable and structured prompt systems using LangChain.

---

## ⚙️ Features Implemented

- PromptTemplate (no hardcoding, reusable prompts)
- Multi-input prompt system (topic, audience, tone)
- Prompt variations (teaching, interview, storytelling)
- ChatPromptTemplate with roles (teacher, interviewer, motivator)
- Input validation layer
- Dynamic prompt generator function
- Template reusability testing

---

## 🧠 Key Concepts

- Prompt Engineering
- Dynamic Prompt Generation
- Input Validation
- Modular Design
- Role-based Prompting

---

## 🔄 Pipeline Flow

User Input  
→ Validation  
→ Prompt Template  
→ Dynamic Prompt Generation  
→ Output
